linear mapping

A homomorphism (function distributive over group operation (addition here)), say, f:V→Wf:V\to Wf:V→W﻿ , between vector spaces is called a linear mapping if it meets this extra criterion : ∀  λ∈R,x∈V,    f(λx)=λf(x)\forall\; \lambda \in \mathbb{R},\bold{x}\in V,\;\;f(\lambda \bold{x}) = \lambda f(\bold{x})∀λ∈R,x∈V,f(λx)=λf(x)﻿ .

Because of the linearity, we can represent such a function as a matrix.

Suppose f:V→Wf:V\to Wf:V→W﻿ is a linear map with V,WV,WV,W﻿ expressed using the basis B={b1,b2… }B=\{\bold{b_1,b_2}\dots\}B={b1​,b2​…}﻿ and C={c1,c2… }C=\{\bold{c_1,c_2}\dots\}C={c1​,c2​…}﻿ , then fff﻿ can be expressed by a matrix with the ithi^{\text{th}}ith﻿ column vector given by the coordinates of f(bi)f(\bold{b_i})f(bi​)﻿ as expressed using C .

Column rank and row rank are the same.

Define column rank to the number of independent columns and row rank to be number of independent rows for any matrix.

Since elementary row operations do not change the column rank or row rank (poof left as exercise), thus after Guassian elimination, these quantities will still be same as of the original matrix.
